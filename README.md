# Agile Week
## Get started
To launch server on rasberry PI :
* Connect with Pi with ssh
* Go to ~/Agile/server/
* run command *make*
* run *./server*

## Useful informations
### Rasberry

**SSH :**
* user : pi
* password : raspberry

**WIFI (AD-HOC)**
* SSID : raspi-wifi
* pwd : raspi123

**ARDUINO**
* Port : /dev/ttyUSB0
* Speed : 115200 baud

commandes :
* Move Roll : MRxxx
* Move Yaw MYxxx
* GetCurrentPosition : G
* Go to Home : H

double (entre  -90 et 90)
